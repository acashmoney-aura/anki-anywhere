# Anki Anywhere

Anki-style flashcards for web + mobile.

## Features

- per-user sync with Convex
- resumable study sessions
- imports from CSV / TSV / JSON / `front::back::tags`
- `Again / Hard / Good / Easy` scheduling aligned to Anki's classic default deck behavior
- optional Google sign-in

## Stack

- React + Vite
- Convex
- Convex Auth
- Framer Motion

## Local setup

```bash
npm install
npx convex dev
npx @convex-dev/auth
npm run dev
```

If Convex Auth is already initialized for your deployment, the auth command will just confirm the setup.

## Environment

Local frontend:

```bash
VITE_CONVEX_URL=your_convex_url
```

Optional Google sign-in:

```bash
AUTH_GOOGLE_ID=your_google_oauth_client_id
AUTH_GOOGLE_SECRET=your_google_oauth_client_secret
VITE_ENABLE_GOOGLE_AUTH=true
```

Notes:
- `AUTH_GOOGLE_ID` and `AUTH_GOOGLE_SECRET` must be set on the Convex deployment.
- `VITE_ENABLE_GOOGLE_AUTH=true` must be set where the frontend builds (for example GitHub Actions).
- You must also add the correct redirect URLs in Google Cloud Console.

## GitHub Pages deploy

This repo includes `.github/workflows/deploy-pages.yml`.

Set these GitHub Actions variables/secrets before deploying:

```txt
VITE_CONVEX_URL=https://<your-deployment>.convex.cloud
VITE_ENABLE_GOOGLE_AUTH=true   # only if Google sign-in is configured
```

If Google sign-in is enabled, also set these on the Convex side:

```txt
AUTH_GOOGLE_ID=...
AUTH_GOOGLE_SECRET=...
JWT_PRIVATE_KEY=...   # generated by `npx @convex-dev/auth`
JWKS=...              # generated by `npx @convex-dev/auth`
```

## Scheduler note

The app now follows Anki's classic default scheduling behavior much more closely:

- learning steps: `1m`, `10m`
- relearning step: `10m`
- graduation: `1d` on Good, `4d` on Easy
- review multipliers: Hard `1.2x`, Easy bonus `1.3x`
- ease changes: Again `-0.20`, Hard `-0.15`, Easy `+0.15`
- lapse handling matches Anki's default minimum 1-day lapse interval plus relearning step

This is intentionally aligned to Anki's classic default deck logic from upstream, not full desktop feature parity for every advanced Anki option.
